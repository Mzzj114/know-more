此项目将会在 2026/6/1 至 2028/9/1 内活跃。共计两年零三个月。
此项目将能被CN和US的用户访问

处于这些需求和项目当前的软件架构，我们提出项目部署的办法。

```
用户 → Cloudflare 域名,CDN,与防护 → AWS EC2 实例(美国西海岸) → nginx docker → django+uwsgi docker ← AWS RDS MySQL
```

AWS 请见 https://us-west-2.console.aws.amazon.com/ec2/
Cloudflare 请见 https://dash.cloudflare.com/

部署步骤：

1. 创建一个有 docker 的 AWS EC2 实例，获得其 IP 地址。nginx 放一个 测试 html 文件
2. 创建一个 Cloudflare 域名，将 IP 地址绑定到域名，创建 Cloudflare CDN 储存静态文件
3. 根据已有信息写 docker 配置。在实例上运行 docker 容器
